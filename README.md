## To run

### *Docker*

Install docker enginer or Docker Desktop

`docker-compose up --build`

*OR*


`docker-compose up`

### *Locally*

1. Install *python 3.10*

2. Install *pipenv*

3. Install *nodejs 19.x*

4. Install *npm* and *yarn*

#### Terminal 1:

`cd app`

`yarn install`

`yarn start`

#### Terminal 2:

`pipenv install`

`pipenv run unit-tests`

`pipenv run start-api`
